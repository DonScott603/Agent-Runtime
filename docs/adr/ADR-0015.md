# ADR-0015: Derivation grammar: declarative-only, closed

Status: ACCEPTED
Date: 2026-07-06
Register: D15

## Context
RFC-0006 D15. Scope derivation runs inside the gate; embedded expression languages put third-party code inside the security boundary.

## Decision
Derivations are field paths (dot, index, [*]) plus at most one transform from a closed set of seven; underivable payloads deny; wildcard paths require every element to satisfy policy.

## Consequences
The D6 discipline applied to manifests. Capabilities needing more expressiveness split their operations instead.
