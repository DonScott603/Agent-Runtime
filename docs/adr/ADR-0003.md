# ADR-0003: Fail closed on unsupported blocks; declared downgrades only

Status: ACCEPTED
Date: 2026-07-06
Register: D3

## Context
RFC-0001 D3. Silent loss of blocks an adapter cannot represent corrupts provider state and replay fidelity invisibly.

## Decision
Unsupported block -> model call fails, unless the adapter manifest declares a deterministic downgrade for that type AND the profile opts in; downgrades are recorded.

## Consequences
Local models fail loudly until downgrades are declared. Every lossy translation is attributable in the log.
