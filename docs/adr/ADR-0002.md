# ADR-0002: Canonical serialization: JCS-style JSON, no floats

Status: ACCEPTED
Date: 2026-07-06
Register: D2

## Context
RFC-0001 D2. Every hash in the system depends on deterministic bytes; float serialization is not deterministic across languages and library versions.

## Decision
RFC 8785-style canonical JSON: sorted keys, UTF-8 NFC, integers only in core; money as integer minor units; binary via blobs.

## Consequences
Human-inspectable logs. kernel/canon is security-critical and cross-checked against an independent implementation (roadmap E5).
