# ADR-0012: Isolation: platform-tiered native primitives + opportunistic containers (a')

Status: ACCEPTED
Date: 2026-07-06
Register: D12

## Context
RFC-0005 D12. Containers-everywhere trades a solvable latency cost for an unacceptable operational dependency; platforms differ in native primitives.

## Decision
Linux full (Landlock+seccomp+netns); Windows v1 best-effort (restricted token + Job Objects + proxy-only egress) with AppContainer parity as first post-v1 milestone and WSL2 as the documented full-tier path; macOS best-effort. Containers escalate `untrusted` workers when a runtime is detected; never required. effect.executed records mechanisms actually applied.

## Consequences
Honest per-platform guarantees; single-binary identity preserved; hardening queue H1/H2 created.
