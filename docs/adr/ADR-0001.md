# ADR-0001: Tool results as content blocks, not a role

Status: ACCEPTED
Date: 2026-07-06
Register: D1

## Context
RFC-0001 D1. The role model decides where the tool_use/tool_result pairing invariant lives and which provider dialects translate losslessly.

## Decision
Roles are {system, user, assistant}; tool results are core.tool_result blocks in user-role messages.

## Consequences
Pairing invariant (I2) is block-local. Adapters targeting role=tool dialects synthesize the role in translation; the canonical form is the superset.
