# ADR-0018: License

Status: OPEN
Date: 2026-07-06
Register: D18

## Context
Retro-licensing outside contributions is painful; the capability manifest format benefits from ecosystem adoption.

## Decision
UNDECIDED. Options: Apache-2.0 (adoption-friendly, patent grant), AGPL (copyleft for a daemon), proprietary-for-now (defer, accept contribution friction). Recommendation: Apache-2.0 if ecosystem adoption of the manifest format is a goal.

## Consequences
Blocks accepting outside contributions; does not block git init or solo development. Decide before the repo is public.
