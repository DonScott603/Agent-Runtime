# ADR-0018: License

Status: ACCEPTED — 2026-07-07 (owner decision)
Date: 2026-07-06
Register: D18

## Context
Retro-licensing outside contributions is painful; the capability manifest format benefits from ecosystem adoption. The repo published to github.com/DonScott603/Agent-Runtime on 2026-07-07, making the decision due.

## Options
Apache-2.0 (adoption-friendly, patent grant), AGPL (copyleft for a daemon), proprietary-for-now (defer, accept contribution friction). Recommendation was Apache-2.0 if ecosystem adoption of the manifest format is a goal.

## Decision
Apache-2.0. Canonical text at /LICENSE (verbatim from apache.org).

## Consequences
Unblocks outside contributions with an explicit patent grant. The appendix copyright boilerplate is left templated; add a NOTICE file naming the copyright owner if attribution is wanted. Per-file license headers are optional under Apache-2.0 and are not used.
