# ADR-0013: Uncertain-effect default: suspend

Status: ACCEPTED
Date: 2026-07-06
Register: D13

## Context
RFC-0005 D13. Crash windows leave effects that may or may not have reached the world; unannotated operations need a default.

## Decision
Manifest field effect_class per operation; operations without a declared class default to suspend-on-uncertain; linter flags missing classes.

## Consequences
A forgotten annotation fails toward a human question, never a double-send. kill -9 matrix (roadmap E7) enforces it.
