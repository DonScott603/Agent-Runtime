# ADR-0010: Message bodies always in blobs

Status: ACCEPTED
Date: 2026-07-06
Register: D10

## Context
RFC-0002 D10. Bulk personal content in event payloads entangles erasure with the hash chain and bloats the fold path.

## Decision
msg.appended and effect.result payloads carry structure and blob refs; text bodies live in (encrypted) blobs.

## Consequences
Uniform erasure; small fast event store; chain verification never touches personal content. Cost: one indirection per read, mitigated by a body cache.
