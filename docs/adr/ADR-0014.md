# ADR-0014: Manifests owner-countersigned at install

Status: ACCEPTED
Date: 2026-07-06
Register: D14

## Context
RFC-0006 D14. The manifest defines scope semantics; trust-on-first-use blesses whatever was present first.

## Decision
Install ceremony renders operations/scopes/resources/trust/slots; owner countersigns the manifest hash; gate honors only countersigned manifests; upgrades render as diffs with widenings highlighted.

## Consequences
Manifest substitution is dead on arrival; silent scope widening across versions is impossible by construction.
