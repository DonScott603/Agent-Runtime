# ADR-0004: Per-run hash chains with anchored Merkle head

Status: ACCEPTED
Date: 2026-07-06
Register: D4

## Context
RFC-0002 D4. One global chain serializes concurrent appends and forces whole-history verification; per-run chains without an anchor make whole-run deletion undetectable.

## Decision
prev_hash chains within each run; global gapless seq for cross-run order; periodic anchor event carrying a Merkle root over run heads, exported/signed off-machine.

## Consequences
Runs verify independently and in parallel. Anchor cadence bounds the undetectable-rewrite window and is a config knob.
