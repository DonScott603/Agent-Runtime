# ADR-0008: Default-ask for granted capabilities

Status: ACCEPTED
Date: 2026-07-06
Register: D8

## Context
RFC-0003 D8. What happens when a granted capability's operation matches no rule.

## Decision
Ungranted -> deny (non-negotiable). Granted with no matching rule -> ask.

## Consequences
Silence is never consent; every allow traces to an explicit rule id. First-week noise is absorbed by narrow approve-always offers.
