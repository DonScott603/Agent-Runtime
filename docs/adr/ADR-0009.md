# ADR-0009: Erasure keys: per-run, wrapped by owner master key

Status: ACCEPTED
Date: 2026-07-06
Register: D9

## Context
RFC-0002 D9. Append-only storage must still support deletion of personal content; retrofitting encryption is a migration.

## Decision
Every blob encrypted at write with a per-run data key wrapped by the owner master key; per-blob erasure by re-wrap exclusion; erasure = key destruction + signed redaction.applied event.

## Consequences
Erasure is uniform, logged, and distinguishable from tampering; granularity is re-wrappable later, encryption-from-day-one is not.
