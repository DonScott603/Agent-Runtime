# ADR-0005: Gapless seq; kernel-mediated time and entropy

Status: ACCEPTED
Date: 2026-07-06
Register: D5

## Context
RFC-0002 D5. Replay determinism requires that ordering and every ambient read be reconstructable from the log.

## Decision
seq assigned gapless on commit by the single log-writer; ts informational; plugins obtain time/entropy via kernel handles, recorded as clock.read/rng.seed events.

## Consequences
Crash recovery reasoning is simple (log ends at a boundary, no gaps). Ambient-state violations are mechanically detectable.
