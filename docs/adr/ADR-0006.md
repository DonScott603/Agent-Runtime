# ADR-0006: Structured scope matchers; closed set of five

Status: ACCEPTED
Date: 2026-07-06
Register: D6

## Context
RFC-0003 D6. String-pattern scopes make parsing the security boundary; hostile qualifier values become grammar.

## Decision
Qualifiers are typed matcher objects: Exact, Prefix, Suffix, OneOf, NumericRange. No general pattern language. Display syntax is a projection.

## Consequences
Matching is typed comparison. Extending the matcher set is an RFC-level event; feature pressure is answered by splitting operations.
