# ADR-0011: Plugin delivery: compile-time pure, subprocess effectful

Status: ACCEPTED
Date: 2026-07-06
Register: D11

## Context
RFC-0004 D11. Go's native plugin package is not viable; pure plugins sit on hot paths; effectful plugins already require processes.

## Decision
Pure plugins are Go modules registered at build time (in-process); effectful plugins are broker-supervised subprocess workers. WASM deferred until third-party pure plugins are in demand.

## Consequences
No IPC tax on context/routing. Pure plugins are kernel-trust at v1 (threat-model R6): no third-party code compiles into the binary.
