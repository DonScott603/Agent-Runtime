# ADR-0017: Stability line: the log is sacred from v0.1

Status: ACCEPTED
Date: 2026-07-06
Register: D17

## Context
versioning.md D17. Code contracts and data contracts age differently; 0.x conventions permit breaking both unless a line is drawn.

## Decision
Persisted/countersigned contracts (canon rules, envelope, message core, shipped payload schemas, chain rules, manifest schema) freeze at first real run and evolve only by upcaster/addition; Go interfaces churn freely until 1.0.

## Consequences
The corpus ratchet makes the line mechanical: no merge may break the past. 1.0 means code-contract freeze, nothing more.
